◆ NoteNewRequest()

J* NoteNewRequest ( const char *  request)

Create a new request object to populate before sending to the Notecard. Lock for mutual exclusion, not only because access to the card must be serialized, but also because both C++ and ArduinoJSON call malloc() which is not a thread-safe operation.

Parameters
requestThe name of the request, for example hub.set.
Returns
a J cJSON object with the request name pre-populated.
