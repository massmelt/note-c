C/C++ Library for the Notecard

Introduction

The note-c C/C++ library for communicating with the Blues Wireless Notecard via serial or I2C.

This library allows you to control a Notecard by writing a C or C++ program,. Your sketch may programmatically configure Notecard and send Notes to Notehub.io.

Dependencies

This library bundles the cJSON JSON parser library.

In addition, this library requires a physical connection to a Notecard over I2C or Serial to be functional.
