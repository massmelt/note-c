note-c

The note-arduino Arduino library for communicating with the Blues Wireless Notecard via serial or IÂ²C.

This library allows you to control a Notecard by writing a C or C++ program. Your program may programmatically configure Notecard and send Notes to Notehub.io.

This library is used by the note-arduino library, which includes it as a git submodule.
